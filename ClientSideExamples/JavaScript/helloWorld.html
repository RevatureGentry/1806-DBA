For exmaple, every line in JavaScript needs to be terminated with a semicolon. However, if we forgete to include one, the JavaScript engine will inject one for us.

There are 6 primative data types in JavaScript. They are

  1. string
  2. boolean
  3. number (64-bit floating point)
  4. null
  5. undefined
  6. Symbol (new in ECMAscript 6)

Template Literal

A template literal is a simplified version of injecting JavaScript expressions into strings. To use a Template Literal, we use the ES6 backticks (``) and inject our JavaScript inside ${}}.

Falsey refers to values that are not the boolean false, but evaluate to false. Values that are falsey include:

  • 0, the number zero
  • "", the empty string
  • null
  • undefined
  • NaN

Note: null and undefined do not have == equality with any other value except each other. NaN is not == or === to anything, including itself.

Truthy is any other value not falsey.
